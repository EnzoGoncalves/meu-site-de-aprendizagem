Formulário


O Elemento <form>


O <form> elemento HTML é usado para criar um formulário HTML para entrada do usuário.


O <form> elemento é um container para diferentes tipos de entrada, como: campos de textos, caixas de seleção. botões de opção, botão de envio, etc.



O Elemento <input>


O elemento HTML <input> é o elemento de formulário mais usado.


Um <input> pode ser exibido de várias maneiras, dependendo do atributo type.


Text Fields


O <input type="text"> define um campo de entrada de linha única para entrada de texto.



O Elemento <label>


A tag <label> define um rótulo para muitos elementos do formulário.


O elemento <label> é útil para usuários de leitores de tela, porque o leitor de tela lerá em voz alta o rótulo quando o usuário focar no elemento de entrada.


O elemento <label> também ajuda os usuários que têm dificuldade em clicar em regiões muito pequenas porque quando o usuário clica no texto dentro do elemento <label>, ele alterna para o radio button / checkbox.


O atributo for da tag <label> deve ser igual ao atributo id do elemento <input> para uni-los.



CheckBoxes


O <input type="checkbox"> define uma checkbox.


As checkboxes permitem o usuário selecionar zero ou mais opçãos de um número limitado de escolhas.
