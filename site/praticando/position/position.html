CSS Layout - A Propriedade Position


A propriedade position especifíca o tipo do método do posicionamento usado para um elemento (static, relative, fixed, absolute or sticky).


Há 5 diferentes valores para position:

  • static
  • relative
  • fixed
  • absolute
  • sticky

Elementos são posicionados usando as propriedades top, bottom, left e right. No entanto, estas propriedades não vão funcionar a menos que a propriedade position seja definida primeira. Eles também trabalham diferente dependendo do valor da position.


position: static;


Elemento HTML são posicionados static por padrão


Elementos com posicionamento static não são afetados pelas propriedades top, bottom, left e right.


Um elemento com position: static; não é posicionado de alguma maneira especial; está sempre posicionado de acordo com o fluxo normal da página


Este elemento div está com position: static;


position: relative;


Um elemento com position: relative; é posicionado em relação à sua posição normal.


Definir as propriedades top, right, bottom e left de um elemento relativamente posicionado fará com qe ele seja ajustado para longe de sua posição normal. Outro conteúdo não será ajustado para caber em qualquer lacuna deixada pelo elemento.


Este elemento possui position: relative;

Foi usado left: 30px;


position: fixed;


Um elemento com position: fixed é posicionado em relação à janela de visualização (viewport), o que quer dizer que sempre está no mesmo lugar mesmo que a página seja scrolada. As propriedades top, right, bottom, left são usadas para posicionar o elemento.


Um elemento fixed não deixa uma lacuna na página onde normalmente estaria localizada.


Observe o elemento fixo no canto inferior direito da página. O CSS que foi usado:
position: fixed; bottom: 0; right: 0; width: 300px; border: 3px solid #73AD21;


Este elemento <div> tem position: fixed;

position: absolute;


Um elemento com position: absolute; é posicionado em relação ao ancestral posicionado mais próximo (em vez de posicionado em relação à janela de visualização, como fixed).


Contudo, se um elemento posicionado de forma absoluta não tiver ancestrais posicionados, ele usa o corpo do documento e se move junto com a rolagem (scroll) da página.


Nota: Um elemento "posicionado" é aquele cuja posição é qualquer coisa exceto static.

Este elemento <div> possui position: relative;

Este elemento <div> possui position: absolute;


Ele pegou a propriedade relative do pai dele e assim pôde usar as propriedades top, right, bottom, left.


position: sticky;


Um elemento com position: sticky; é posicionado baseado na posição de rolagem (scroll) do usuário.


Um elemento sticky alterna entre relative e fixed, dependendo da posição da rolagem. É posicionado relative até que uma determinada posição de deslocamento seja encontrada na janela de exibição - então, ele "se fixa" no lugar (como a posição: fixed).
