CSS Layout - A Propriedade Display

A propriedade display é a propriedade mais importante do CSS para controlar o layout.


The display Property

A propriedade display especifica se/como um elemento é exibido.


Cada elemento HTML tem um valor de exibição padrão dependendo de que tipo de elemento é ele. O valor de padrão para a maioria dos elementos é block ou inline.

Block-level Elements

Um elemento nível de bloco sempre começa em uma nova linha e ocupa toda a largura disponível (se estende para a esquerda e para a direita o máximo que pode).

O elemento <div> é um elemento de nível de bloco.

Exemplos de elementos de nível de bloco:


  • <div>
  • <h1> - <h6>
  • <p>
  • <form>
  • <header>
  • <footer>
  • <section>

Inline Elements

Um elemento inline não começa em uma nova linha e só ocupa a largura necessária

Este é um elemento <span> embutido dentro de um parágrafo.


Exemplos de elementos inline:


  • <span>
  • <a>
  • <img>

Display: none;

display: none; é comumente usado com JavaScript para ocultar e mostrar elementos sem deletá-los ou recriá-los. Dê uma olhada no nosso último exemplo nesta página se você quer saber como isto pode ser alcançado


O <script> elemento usa display: none; como padrão.


Override (substituir) o display padrão de um elemento

Conforme mencionado, cada elemento tem um valor de exibição padrão. No entanto, você pode substituir isso.


Alterar um elemento embutido para um elemento de bloco, ou vice-versa, pode ser útil para fazer a página parecer de uma maneira específica e ainda seguir os padrões da web.


Um exemplo comum é criar <li> elementos embutidos para menus horizontais:

  • Home
  • Sobre
  • Contato
Nota: Definir a propriedade display de um elemento muda apenas como o elemento é exibido, NÃO que tipo de elemento ele é. Portanto, um elemento embutido com display: block; não pode ter outros elementos de block dentro dele.

Hider an Element - display:none or visibility:hidden?


display: none

visibility: hidden

Reset

Esconder um elemento pode ser feito configurando a propriedade display para none. O elemento estará escondido, e a página exibirá como se o elemento não estivesse lá.


visibility:hidden; também esconde um elemento.


No entando, o elemento ainda vai estar ocupando o mesmo espaço de antes. O elemento estará escondido, mas ainda afentando o layout da página, como vimos no exemplo acima.
