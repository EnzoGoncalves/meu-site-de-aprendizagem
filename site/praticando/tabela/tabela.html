Tabela

As tabelas HTML permitem que os desenvolvedores da web organizem os dados em linhas e colunas.


A <table> tag define uma tabela HTML.


Cada linha da tabela é definida com uma <tr> tag. Cada cabeçalho da tabela é definido com uma <th> tag. Cada dado / célula da tabela é definido com uma <td> tag.


Por padrão, o texto no elemento <th> está em negrito e centralizado.


Por padrão, o texto no elemento <td> é regular e alinhada à esquerda.


Exemplo de uma tabela:


Música Jogos Comida
Circle Fortnite Sushi Hambúrguer
Eletronica Rocket League Pizza
Psycho Fortnite Lasanha
