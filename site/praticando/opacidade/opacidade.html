CSS Opacidade / transparência


A propriedade opacity especifica a opacidade / transparência de um elemento.



Transparência de uma imagem


A propriedade opacity pode ter o valor de 0.0 - 1.0. Quanto menor o valor, mais transparente:


opacity: 0.2

opacity: 0.5

opacity: 1 (default)




Efeito de transparência ao passar o mouse por cima


A propriedade opacity é frequentemente usada com o seletor hover para mudar a opacidade ao passar o mouse por cima:





Box transparente


Quando usamos a propriedade opacity para adicionar transparência de fundo de um elemento, todos os filhos do elemento herdarão a mesma transparência. Isto pode fazer o texto dentro do elemento totalmente transparente e difícil de ler:

1

opacity
0.6

0.3

opacity
0.1




Transparência usando RGBA


Caso você não queira aplicar a opacidade para os elementos filhos, como no exemplo acima, use valores de cor RGBA. No próximo exemplo configure a opacidade para a cor de fundo e não para o texto:

Um valor de cor RGBA é especificado com: rgba(red, green, blue, alpha). O parâmetro alpha é um número entre 0.0 (totalmente transparente) e 1.0 (totalmente opaco).
