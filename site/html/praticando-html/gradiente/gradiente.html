CSS gradientes

Gradient Backgrounds

Gradientes te deixam exibir transições suaves entre duas ou mais cores especificadas.


CSS define dois tipos de gradientes:


  • Gradiente linear (vai para baixo/cima/esquerda/deireita/diagonalmente)
  • Gradientes radiais (definido pelo seu centro)

Gradiente Linear

Para criar um gradiente linear é preciso definir ao menos duas interrupções de cor. As paradas de cor são as cores entre as quais você deseja renderizar transições suaves. Você também pode definir um ponto inicial e uma direção (ou um ângulo) junto com o efeito de gradiente.


Sintaxe:

background-image: linear-gradient(direction, color-stop1, color-stop2, ...);

Direção - de cima para baixo (este é o padrão)

Direção - Diagonal

Você pode fazer um gradiente diagonalmente especificando as posições iniciais horizontal e vertical.

Usando ângulos

Se você quiser ter mais controle sobre a direção do gradiente, pode definir um ângulo, em vez das direções predefinidas. Um valor de 0deg é equivalente a "to top". Um valor de 90deg é equivalente a "à direita". Um valor de 180deg é equivalente a "to bottom".


Sintaxe:

background-image: linear-gradient(angle, color-stop1, color-stop2);

Usando tranparência

Gradiente também suporta transparência, o que pode ser usado para criar efeitos de desbotamento.


Para adicionar transparência, usamos a função rgba() para definir a parada de cor. O último parâmetro na função rgba() pode ter um valor de 0 até 1, e é definido a transparência da cor: 0 indica totalmente transparente, 1 indica totalmente com cor (sem transparência).
