CSS Layout - float and clear

A propriedade CSS float especifica como um elemento deve flutuar.


A propriedade CSS clear especifica quais elementos podem flutuar ao lado do elemento limpo e em que lado.

A propriedade float

A propriedade float é usada para posicionar e formatar o conteúdo, por exemplo, deixar uma imagem flutuar à esquerda do texto em um contêiner.


A propriedade float pode ter um dos seguintes valores:


  • left - O elemento flutua para a esquerda do contêiner
  • right - O elemento flutua para a direita do contêiner
  • none - o elemento não flutua (será exibido apenas onde no texto ocorre). Este é o valor padrão
  • inherit - O elemento herda o valor float do pai (parent)

A propriedade clear

A propriedade clear especifica qual elemento pode flutuar atrás ao lado do elemento limpo e em qual lado.


A propriedade clear pode ter um dos seguintes valores:


  • none - permite elementos flutuantes em ambos os lados. Este é o valor padrão.
  • left - nenhum elemento flutuante permitido no lado esquerdo
  • right - não são permitidos elementos flutuantes no lado direito.
  • both - nenhum elemento flutuante permitido no lado esquerdo ou direito
  • inherit - o elemento herda o valor clear de seu pai (parent)

A maneira mais comum de usar a propriedade clear é depois de usar a propriedade float em um outro elemento.


Ao limpar os floats, você deve combinar o clear com o float: Se um elemento for flutuado para esquerda, você deve limpa para a esquerda. Seu elemento continuará a flutar, mas o elemento clear aparecerá abaixo dele na página web.

Clearfix

Neste exemplo, a imagem é mais alta que o conteúdo do elemento é, e está flutuando, então está ultrapassando para fora do conteiner:


Lorem ipsum dolor sit amet, consectetur adipiscing elit. Phasellus imperdiet, nulla et dictum interdum...

Adicione uma classe clearfix com overflow: auto; para o conteúdo do conteiner, para arrumar o problema:


Lorem ipsum dolor sit amet, consectetur adipiscing elit. Phasellus imperdiet, nulla et dictum interdum...

Então podemos adicionar overflow: auto; ao elemento que o contém para corrigir este problema.


O overflow: auto funciona bem desde que você consiga manter o controle de suas margens e paddins (caso contrário, você pode ver barras de rolagem).
