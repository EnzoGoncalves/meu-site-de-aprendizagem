CSS Transformações 2D

Transformações CSS te permitem rotacionar, escalar, e enviesar (deixar fora de quadro) os elementos.


Neste capítulo você irá aprender sobre a propriedade CSS transform.

Métodos de Transformações 2D

Com a propriedade transform você poderá usar os seguintes métodos de transformação 2D:


  • translate()
  • rotate()
  • scaleX()
  • scaleY()
  • scale()
  • skewX()
  • skewY()
  • skew()
  • matrix()

translate()

O método translate() move um elemento de sua posição atual (de acordo com os parâmetros dados ao eixo-X e eixo-Y).


No exemplo a seguir o elemento <div> se move 100px para a direita, e 20px para baixo.


rotate()

O método rotate() rotaciona um elemento sentido-horário ou sentido anti-horário de acordo com os graus dados.


No exemplo a seguir o elemento <div> rotaciona para o sentido-horário 45 graus:



Usando números negativos para especificar o grau da rotação, irá rotacionar o elemento no sentido anti-horário.

scale()

O método scale() aumenta ou diminui o tamanho do elemento (de acordo com os parâmetros dados ao width e height).


No seguinte exemplo, o elemento <div> aumentará de tamanho duas vezes da sua largura original, e três vezes da sua altura original.
