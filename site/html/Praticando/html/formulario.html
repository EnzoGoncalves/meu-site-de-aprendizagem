Formulário


O <form> elemento

O <form> elemento HTML é usado para criar um formulário HTML para entrada do usuário.


O <form> elemento é um container para diferentes tipos de entrada, como: campos de textos, caixas de seleção. botões de opção, botão de envio, etc.


O <input> elemento

O <input> elemento HTML é o elemento de formulário mais usado.

Um <input> pode ser exibido de várias maneiras, dependendo do type atributo


Text Fields

O <input type="text"> define um campo de entrada de linha única para entrada de texto


The <label> Element

A <label> tag define um rótulo para muitos elementos do formulário.

O <label> elemento é útil para usuários de leitores de tela, porque o leitor de tela lerá em voz alta o rótulo quando o usuário focar no elemento de entrada.

O <label> elemento também ajuda os usuários que têm dificuldade em clicar em regiões muito pequenas porque quando o usuário clica no texto dentro do <label> elemento, ele alterna para o radio button / checkbox

O for atributo da <label> tag deve ser igual ao id atributo do <input> elemento para uni-los.


CheckBoxes

O <input type="checkbox> define uma checkbox.

As checkboxes permitem o usuário selecionar zero ou mais opçãos de um número limitado de escolhas
