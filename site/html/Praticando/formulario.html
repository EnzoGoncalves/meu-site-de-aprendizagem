Formulário


O <form> elemento

O <form> elemento HTML é usado para criar um formulário HTML para entrada do usuário.


O <form> elemento é um container para diferentes tipos de entrada, como: campos de textos, caixas de seleção. botões de opção, botão de envio, etc.


O <input> elemento

O <input> elemento HTML é o elemento de formulário mais usado.

Um <input> pode ser exibido de várias maneiras, dependendo do type atributo


Text Fields

O <input type="text"> define um campo de entrada de linha única para entrada de texto
